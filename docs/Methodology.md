## 🔍 Metodologia

A análise foi estruturada em cinco etapas principais, utilizando BigQuery para processamento e modelagem dos dados e Power BI para análise e visualização:

### BigQuery
- Limpeza e preparação dos dados;  
- Criação de novas variáveis;  
- Modelagem de dados com uso de JOINs.

### Power BI
- Análise exploratória de dados; 
- Validação de hipóteses por meio de análise de correlação. 

Essa abordagem garantiu consistência dos dados e confiabilidade na geração dos insights.
